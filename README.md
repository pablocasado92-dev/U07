# UD07 - Programación en Java

Repositorio para las actividades de la Unidad Didáctica 07.

## 📁 Estructura

Las actividades están organizadas en carpetas:

- `A00` - Práctica de herencia, superclase y subclase
- `A01` - Diseño de una clase Hora con metodos propios
- `A02` - Implementación de la clase HoraExacta
- `A03` - Implementación metodo equals() en HoraExacta
- `A04` - Clase abstracta Instrumento, que almacena en una tabla las notas musicales de una melodía.
- `A05` - clase Piano heredando de la clase abstracta Instrumento.

*(Cada carpeta contiene sus archivos `.java` correspondientes)*
